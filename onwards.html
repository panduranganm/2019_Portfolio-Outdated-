Final Project for Intro to Programming

Onwards

Onwards is a 3D game developed with Panda3D where users can walk through infinite, automatically generated landscapes. Users can enjoy a calm, immersive experience where they can control an avatar and interact with their surroundings as though the are actually walking through a forest, desert, or grassland. I brainstormed and developed this project by myself over the course of 3 weeks.

01

Week 1 - Brainstorming

- developed game idea
- designed Blendr objects (trees, ground, boxes, etc)
- designed start screen - made objects responsive to collisions with other objects

02

Week 2 - Developed Game

- implemented the three different landscapes
- made scene automatically generate as user walks
- added ways for character to interact with environment (cut/place objects, view area from aerial view)

03

Week 3 - Added Features

- added mini game (find the chicken!)
- made data storage more efficient (so that game doesn't slow down as more elements are added)
